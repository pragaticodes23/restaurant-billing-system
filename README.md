# Restaurant Bill & Discount System

A beginner Python project that calculates restaurant bills and applies discounts based on the total bill amount, membership status, and weekend offers.

🚀 Features

* Takes customer details as input
* Calculates bill-based discounts automatically
* Provides additional discounts for members
* Provides extra discounts on weekends
* Calculates total discount amount
* Displays the final payable bill
* Shows a detailed discount breakdown

🧠 Concepts Used

* Variables
* User Input
* Strings
* Arithmetic Operators
* Comparison Operators
* Conditional Statements (if-elif-else)
* Decision Making

Input-
